Math Class

The Math Class

May 11, 2019

All methods in the Math class are static methods.

What this means is that they do not need to be called using an object, just the class name(Liang, p130)

className.methodName(arguments)

instead of

objectName.methodName(arguments)

Example: Math.random(); Math.pow(a,b); Math.max(a, b);

There is no need to import the Math class since it is implicitly imported as part of the java.lang package (Liang, p124).

Here are the many methods that come with the Math class as well as more information on the Math class: "https://docs.oracle.com/javase/8/docs/api/java/lang/Math.html"

The Math class also has two constants that can be used : PI and E (Liang, p120).

Example: double area = radius * radius * Math.PI;
