The different data types

Primitive Types vs Reference Types

May 11, 2019

When passing primitive data types as arguments, whatever is being done to it within the method will not affect it outside the method. The value of the variable is not affected outside the method.

Ex(Liang, p214):

Ouput:
Before the call, x is 1
n inside the method is 2
After the call, x is 1

When passing reference typesas arguments, whatever is being done to it within the method WILL affect it outside that method. So if you pass an object to a method, it will be affected outside that method.
